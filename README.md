# VozClin Landing

Landing page premium do VozClin, um SaaS de apoio à documentação clínica por voz com IA e revisão humana obrigatória.

## Posicionamento

O VozClin transforma ditados, áudios ou conversas clínicas em fichas estruturadas e revisáveis para apoiar a rotina de clínicas e profissionais de saúde.

Limites de comunicação:

- Não é prontuário eletrônico.
- Não faz diagnóstico.
- Não prescreve.
- Não substitui médico, fisioterapeuta, dentista, recepção ou equipe humana.
- Exige revisão humana antes de qualquer uso.

## Stack

- Next.js 16
- React 19
- TypeScript
- CSS global próprio
- Lucide React para ícones
- Pronto para importação na Vercel

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm run build
```

- `app/page.tsx`: landing e componentes visuais.
- `app/globals.css`: direção visual, responsividade e microinterações.
- `lib/site-content.ts`: conteúdo editável da página.
- `public/`: assets visuais e favicon.
